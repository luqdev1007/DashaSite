// 01 — 概念

Core Concept

Рюичи — курьер пиццерии «Хот Катана», которая служит прикрытием для клана Якудза. Эпоха: феодальная Япония с нино-панк эстетикой. Каждая доставка — секретная миссия. Каждая секунда промедления стоит репутации клана.
Core Fantasy: Игрок — городской призрак. Движение = оружие. Каждый прыжок, удар, бросок кунай — не отдельное действие, а звено в цепи моментума. Уровень — не препятствие, а инструмент. Враги — не стены, а трамплины.
Жанр
2D Turbo Platformer

Action-платформер с акцентом на скоростное прохождение уровней. Ближайшие референсы по feel: Celeste (движение), Katana ZERO (импакт), Mark of the Ninja (стелс-момент).

Платформа
Web (Browser)

Unity WebGL билд. Приоритет на мгновенный старт без установки. Оптимизация под 60fps в браузере. Управление: клавиатура + геймпад.

MVP Scope
3–5 уровней

Прототип для проверки core-механик. Один мир (Shibuya-inspired). Полный набор движения, базовые враги, таймер-система, лидерборд.

// 02 — 力学

Арсенал Героя

Боёвка здесь не самоцель — каждое действие одновременно является инструментом перемещения. Убил врага — получил трамплин. Бросил кунай — получил точку притяжения. Система строится вокруг одного принципа: моментум никогда не прерывается.
①
Movement
Прыжок / Двойной прыжок

Вариативная высота в зависимости от удержания кнопки. Второй прыжок доступен в воздухе с кувырком. Wall-jump от вертикальных поверхностей. Ключевое: двойной прыжок обновляется при попадании атаки по голове врага (кроме шипастых). Позволяет строить бесконечные воздушные цепочки через головы противников.

SPACE SPACE × 2 ↑ + WALL
②
Movement
Дэш / Рывок

Горизонтальный рывок с кратковременной неуязвимостью. В воздухе — воздушный дэш (меняет направление вектора). При рывке сквозь врага — наносит урон и сохраняет скорость. Кулдаун сбрасывается при приземлении.

SHIFT ↔ + SHIFT
③
Movement + Combat
Кунай / Верёвка / Гарпун

Три режима в одном:
Гарпун к точке — кунай цепляется за уступ или стену, игрок притягивается к нему с инерцией маятника. Отпустить в нужный момент = баллистический прыжок с сохранением скорости.
Гарпун к врагу — притягивает летящего/стоящего врага к игроку ИЛИ притягивает игрока к врагу (переключается кнопкой). При подлёте — автоудар с усилением от моментума.
Бросок-снаряд — летит по дуге, оглушает при попадании в голову на 2 секунды.

RMB / Q Q + CTRL = реверс
④
Movement + Combat
Ближний бой

Быстрая атака ближнего радиуса. При ударе по врагу — игрок получает отскок в противоположном направлении от удара (как от стены). Это ключевой инструмент изменения вектора в воздухе без потери скорости. Удар снизу-вверх (во время прыжка вниз) = slam, враг улетает вверх + игрок подбрасывается.

LMB / Z ↓ + Z = Slam
⑤
Movement
Скольжение / Паркур

Слайд при беге — снижает хитбокс, сохраняет скорость, можно скользить под низкими препятствиями. При переходе в прыжок из слайда — бонус к дальности прыжка. Wall-slide — медленное скольжение по стене с возможностью оттолкнуться. На наклонных поверхностях — автоускорение вниз.

↓ + RUN ↑ + WALL
⑥
Special
Дымовая бомба

Создаёт облако на 3 секунды. Скрывает от врагов, блокирует лазерные сенсоры, замедляет снаряды пролетающие сквозь. Ограниченный запас (1–3 в зависимости от уровня). В облаке дыма движение героя не производит звука — агрессивные враги теряют цель.

// 03 — 軸

Y-Axis System

Ключевая уникальная механика игры. Действия героя меняются в зависимости от направления и скорости движения по оси Y. Это создаёт физически читаемую боёвку, где каждое действие логично вытекает из состояния движения.
⬆️
Движение вверх

Апперкот при ударе — враг улетает вверх, игрок получает дополнительный импульс. Кунай при броске вверх = дальше летит. Прыжок на ускорении = выше.

⏸️
Нейтраль / земля

Стандартные атаки с горизонтальным нокбэком. Слайд доступен. Базовый бросок кунай. Накопление разгона для следующего прыжка.

⬇️
Падение / движение вниз

Slam-удар при атаке вниз — AoE волна при приземлении, чем выше падение, тем шире волна. Обновление двойного прыжка при попадании по голове врага. Скорость падения = урон slam.

Скорость влияет на урон: Чем быстрее герой движется в момент удара (по любой оси) — тем сильнее нокбэк врага и выше отскок самого героя. Максимальный моментум = максимальная дальность полёта после удара. Это стимулирует игрока постоянно поддерживать скорость.
Кунай + притяжение + удар: Подтянуться гарпуном к летящему врагу → скорость сближения = скорость падения → slam в голову при контакте → обновление double jump → оттолкнуться и использовать второй прыжок уже в воздухе. Полная цепочка без касания земли.
// 04 — 技

Movement Combos

Комбо не требуют заучивания — они вытекают из физики. Опытный игрок обнаруживает их органически. Ниже — задокументированные цепочки для level design и балансировки.
Голова-рикошет
JUMP → ↓ATTACK на голову → JUMP

Базовое обновление double-jump через врага. Позволяет прыгать по цепочке голов через всю комнату. Шипастые враги разрушают цепочку — они запрещённый элемент в этом паттерне. Основа для большинства speedrun маршрутов.

Кунай-маятник
JUMP → KUNAI (стена) → раскачка → RELEASE

Кунай в стену, раскачка на верёвке с набором скорости, отпустить в верхней точке = баллистический запуск. Инерция сохраняется полностью. Позволяет перепрыгивать огромные горизонтальные секции без платформ.

Гарпун-трамплин
KUNAI (враг) → притяжение → SLAM → JUMP

Гарпун к врагу → притянуться к нему → в момент контакта slam вниз → враг летит вниз, игрок отскакивает вверх + обновляет double jump. Если враг летит в других врагов — можно продолжить цепочку.

Слайд-запуск
RUN → SLIDE → JUMP в конце

Скольжение сохраняет скорость бега и добавляет бонус к дистанции прыжка при переходе. Идеально для длинных горизонтальных секций с зазорами. Основной способ пройти через группу врагов без боя.

Воздушный реверс
AIR DASH → KUNAI (обратное направление)

Воздушный дэш вперёд → в воздухе гарпун назад в стену → резкое изменение направления без потери скорости. Позволяет огибать препятствия и уходить от снарядов с сохранением моментума.

Slam-волна
Высокое падение → ↓ATTACK у земли

Чем выше падение, тем больше AoE волна при slam. С максимальной высоты волна накрывает всю маленькую комнату. Отдельные уровни будут содержать секции, спроектированные специально под этот приём.

// 05 — 時間

Timer System

Таймер — не просто ограничение, а основная игровая валюта. Время можно выигрывать и терять. Это создаёт постоянное напряжение без хардкорного punishment — можно рискнуть и заработать время, или сыграть осторожно и потерять.

// 06 — 世界

Уровни MVP

MVP — 3–5 уровней в одном мире. Стилистика: феодальная Япония времён ниндзей, лоу-поли мультяшная эстетика. Каждый уровень вводит новую механику взаимодействия со средой.
一
Рынок Эдо
Lvl 01 · Tutorial

Дневной рынок, торговые ряды, низкие крыши, толпа. Обучающий уровень — широкие переходы, редкие враги, видимые маршруты. Вводит: бег, прыжок, кунай, первый патруль.

Фонарные столбы Толпа (замедляет) Бамбуковые навесы 1 патруль
二
Крыши замка
Lvl 02 · Vertical

Черепичные крыши, уступы, вертикальная секция с wall-jump. Вводит: двойной прыжок, слайд, первый патруль с алертом. Ночное время — дроны-фонарщики по паттерну.

Wall-jump секция Скользкие крыши (дождь) Фонарщик-дрон Скрытый проход
三
Подземный канал
Lvl 03 · Combat

Тёмные водные каналы под городом. Узкие коридоры, много вертикали. Вводит: боёвку как транспорт, цепочку прыжков по головам, slam-волну. 2–3 типа врагов одновременно.

Водный поток (сносит) Бочки-взрывчатки Темнота (дымовая бомба) Цепочка голов ×5
四
Башня Якудзы
Lvl 04 · Mixed

Внутри деревянной башни — лестницы, движущиеся платформы-лифты, охрана. Вводит: гарпун-комбо с врагами, первый мини-босс (Охранник Ямато). Проверка всех механик вместе.

Лифт-платформы Мини-босс Сигнальный барабан (тревога) Гарпун-маршрут
五
Мост через пропасть
Lvl 05 · Speed

Длинная горизонтальная секция с минимумом платформ. Кунай-маятники обязательны. Погоня — Ниндзя-соперник преследует с секунды 0. Финальная проверка скорости и всех механик.

Обрыв (мгновенная смерть) Ниндзя-преследователь Ветер (влияет на прыжок) Speedrun-секция
// ИНТЕРАКТИВНЫЕ ЭЛЕМЕНТЫ УРОВНЯ
Movement
🏮 Фонарный столб

Раскачка на цепи — физика маятника. Набери скорость и улети. Обязателен для некоторых секций — без раскачки не перепрыгнуть.

Hazard / Boost
💨 Воздушный поток

Снизу — подбрасывает вверх (буст). Сбоку — сносит. В паре с кунаем создаёт комбо-маршруты.

Shortcut
🪟 Бумажные стены (сёдзи)

Пробиваются при спринте. Скрытые комнаты, срезки через здания. Всегда даёт бонусное время.

Hazard
🥷 Сигнальный барабан

Если враг добирается до него и бьёт — тревога. Уничтожить кунаем или опередить. Таймер тревоги сжигает секунды.

Environment
💣 Пороховые бочки

Кунай → детонация → цепная реакция. Убивает врагов в радиусе + разрушает стены + подбрасывает героя если он в зоне.

Secret
🐈 Кошка клана

Редкий NPC. Взаимодействие → ведёт к секретному ходу (+5s и альтернативный маршрут). Убегает через 2 секунды.

// 07 — 敵

Бестиарий MVP

Патруль-Гэнин
Patrol · Базовый

Ходит по маршруту, конус зрения. При контакте — тревога. Медленная реакция. Идеален для головы-рикошет цепочки. Slam сверху = мгновенное устранение.

Слабость: Удар по голове, slam, кунай в затылок. Пицца в лицо = 3с слепоты.
Лучник-Кюдока
Ranged · Дальний

Стоит на позиции, стреляет стрелами с паузой. Долгая перезарядка. Мёртвая зона снизу — не видит под собой. Стрелы можно отбить ударом в воздухе.

Слабость: Зайти снизу + комбо. Дымовая бомба = потеря цели навсегда (убегает).
Броня-Самурай
Tank · Тяжёлый

Медленно идёт, не реагирует на обычный удар спереди. Блокирует узкие проходы. Атакует мечом широким замахом. При атаке сзади — уязвим и падает шумно.

Слабость: Кунай в спину, взрыв бочки рядом. Slam сверху = оглушение на 3с.
Дрон-Фонарщик
Aerial · Воздушный

Летит по паттерну, луч фонаря. Тревога при обнаружении. Взрывается с задержкой при уничтожении — можно использовать как мину для соседних врагов.

Слабость: Кунай × 1 = уничтожен (+1s бонус). Дымовая бомба = слепой на 5с.
Ниндзя-Соперник
Elite · Преследователь

Появляется в Level 5. Использует те же механики что и игрок — wall-jump, двойной прыжок, дэш. Преследует непрерывно. Если догоняет — большой штраф по времени. Не умирает, только временно останавливается.

Слабость: Дымовая бомба = 8с стопа. Slam в голову = откидывает далеко назад.
// 08 — メタ

Мета Геймплей

🏆
Лидерборд

Таблица лучших времён по каждому уровню. Глобальная + локальная (среди друзей). Ранг обновляется при каждом прохождении. Основной реиграбельности-крючок для speedrun-аудитории.

⚔️
Прокачка героя

Между уровнями — выбор 1 из 3 апгрейдов. Не RPG-прокачка, а одноразовые модификаторы. Примеры: «+1 дымовая бомба», «дэш сбрасывает кулдаун при убийстве», «slam-волна шире». Сбрасывается между сессиями.

🥷
ИИ-Боты (Сайд)

Нанять ботов-курьеров клана. Дать имя, снарядить. Отправить на авто-миссию. Боты периодически смешно фейлят на упрощённых версиях уровней. Можно смотреть «запись» их провалов. Никак не блокирует основной прогресс.

📜
Нарратив между уровнями

Короткие диалоги (2–4 реплики) в стиле визуальной новеллы. Рюичи и персонажи клана. Раскрывают что в коробке с пиццей. Тон: абсурдный юмор + якудза-серьёзность. Не блокирует геймплей.
